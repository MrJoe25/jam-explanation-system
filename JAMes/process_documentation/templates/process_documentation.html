Model Development Steps

  1. Upload Data: Click on "Develop New Model" and navigate to the "Personal Upload" page to upload a CSV file containing company data based on a template that can be downloaded on the same page.
  2. Select User Data: Choose which uploaded files to look at based on the user.
  3. Select Model Data: Choose which uploaded file to use as the basis for the model.
  4. Data Overview: Examine an overview of the data and clean it if necessary.
  5. Select Relevant Financial Features: Select at least three relative financial features to include in the prediction.
  6. Data Segmentation: Inspect the resulting dataset that contains relative values and choose whether the dataset is to be separated into different segments based on the total assets of the companies included. If so, proceed with the separation.
  7. Select Model Settings: Choose whether Bayesian optimization should be used in model training or which oversampling technique is used, and select the train-test-validation split.
  8. Train Model: Inspect the resulting data and start the model training (when Bayesian optimization was not selected) with choices for adjusting the values for the number of estimators, learning rate, and maximum depth.
  9. Assess Model: Assess the model based on explainability features from XGBoost and SHAP values, as well as evaluation metrics for machine learning models, and export the data if desired.
  10. Assess Predictions: Assess the predictions for individual companies, the companies that are actually bankrupt and their prediction results, the actual status of companies that have been predicted as bankrupt, and companies that were predicted as bankrupt correctly.
